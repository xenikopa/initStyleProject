# Init Style Project

This repository contains various styles to get starting with project markup.

Stack:
- `gulp` - build project
- `pug` - write html
- `styl` - write styles.


## Installation

Install with [npm](http://www.npmjs.com/):

```sh
$ npm install
```
or with [yarn](https://yarnpkg.com/)
```sh
$ yarn install
```

## Developing

Run develop with command
>`yarn run start:dev`

Get prod builded files with command
>`yarn run build`
